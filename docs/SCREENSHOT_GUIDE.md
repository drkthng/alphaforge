# Screenshot Guide

To add screenshots to the README:

1. Launch AlphaForge with some real data ingested.
2. Take screenshots using Windows Snipping Tool (`Win+Shift+S`).
3. Save to `docs/screenshots/` with these exact filenames:

| Screenshot | Filename | What to capture |
| --- | --- | --- |
| Pipeline | `pipeline.png` | Kanban board with strategies in multiple statuses |
| Leaderboard | `leaderboard.png` | Table with 10+ runs, color coding visible |
| Strategy Detail | `detail.png` | Full page with left panel and right panel |
| Equity Curves | `equity_curves.png` | 2-3 runs overlaid, drawdown subplot visible |
| Heatmap | `heatmap.png` | 2D parameter heatmap with color variation |
| Inbox | `inbox.png` | Capture form and some items listed |

4. Commit: `git add docs/screenshots/ && git commit -m "docs: add screenshots"`
